This is a frontend-only multi-restaurant food delivery platform designed as a school project. The website is fully in Arabic (RTL) and uses Libyan Dinar (د.ل) for pricing. It allows users to browse restaurants, view menus with categorized items, and read customer reviews.

Key Features:

Responsive HTML & CSS layout with semantic structure

Right-to-left (RTL) support for Arabic

Restaurant listings with ratings, cuisine type, and delivery time

Menu sections with categories and realistic menu items

Static customer reviews for each restaurant

Minimal JavaScript to toggle between restaurant views and show/hide reviews

Single-page navigation with sections for Home, Restaurant Details, and static informational pages (About, Help, Privacy)

Fully frontend-based, no backend or databases required

Technologies Used:

HTML5

CSS3 (Flexbox & Grid)

Minimal JavaScript for interactivity
